Go has three types of output functions

* println():
this function is similar to print() with the difference that a whitespace is added between the arguments and a newline is added at the end.


* print():
this function prints its arguments with their default format.


* printf():
this function first format its argument based on the giving formatting verb and prints them.

%v is used to print the value of the argument.
%T is used to print the value type of the argument.
